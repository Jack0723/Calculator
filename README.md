# Git Notes

// 初始化 git
git init

// 註冊遠端的 GitHub Repository
git remote add origin https://xxxxxx

// 新增檔案到 Git
git add [FILE_NAME]
git add . // 全部檔案

// 提交
git commit -m "COMMIT_MESSAGE"

// 推送到遠端 GitHub Repository
git push [遠端名稱] [分支名稱]
git push origin main

// 查看 Git 狀態
git status

// 查看遠端名稱
git remote

// 查看分支
git branch

// 建立分支
git branch [分支名稱]

// 切換分支
git checkout [分支名稱]

// 複製遠端 Repository
git clone https://xxxxx

// 拉下遠端分支
git pull origin main
